RestoBook - Система онлайн-бронирования столиков 
Полуфункциональный веб-сайт системы онлайн-бронирования столиков для ресторана. Проект включает в себя четыре основные страницы с различным функционалом джля клиентов и администраторов.


Быстрый старт


Требования:
1. Веб-браузер (Chrome, FIreFox, Safari)
2. Любой текстовый редактор (VSCode, Notepad++)


Установка и запуск:
1. Клонируйте репозиторий
2. Откройте папку проекта в проводнике
3. Распакуйте архив и окройте файл index.html
